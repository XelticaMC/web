
# 利用規約

この規約は、XelticaMC (以下、本サーバーと称する）をご利用いただく上で守っていただく必要のあることが記されています。

長いように思うかもしれませんが、全て重要なこととなっており、本サーバーのプレイヤーはみな本規約を把握していることを前提としています。

なるべく読みやすく書いていますので、読んでいただくようお願いします。

## 定義


- BAN とは、本サーバーへのログインを技術的に禁ずる処罰を指します。
- 警告とは、本サーバーのモデレーターから提示される警告文を指します。
- ミュートとは、本サーバーのチャットに書き込めなくなる処罰を指します。
- キックとは、本サーバーから強制的にログアウトさせる処罰を指します。

<hr/>

- 処罰とは、BAN、警告、ミュート、キックのいずれかを指します。
- 第一処罰、第二処罰、...、第n処罰とは、複数の処罰を持つ規約において、優先順位順に処罰を並べた表現です。
	- 第一処罰から順番に実行されますが、処罰が何らかの理由で実行できなかったり、処罰が意味を為さなかったりした場合に次の処罰が実行されます。
	- 例えば、第一処罰を警告、第二処罰をBANとする場合、第一警告である警告が不可能な場合、無視した場合に第二処罰のBANが実行されます。
- 処罰のスキップとは、何らかの理由によって処罰を中断することを指します。複数の処罰がある場合、次の処罰が実行され、次の処罰が無い場合はそれ以上の処罰は行われません。

<hr/>

- プレイヤーとは、本サーバーに参加する者を指します。
- オーナーとは、本サーバーの最も高い権限を持つプレイヤーを指します。

<hr/>

- 資産とは、アイテム、建築、土地、エビパワー等を指します。
- 所有権とは、特定の資産を所有できる権利を指します。共有資産の場合は全ての人が平等に、個人資産の場合は特定の所有者およびそれらによって許可された人が持ちます。
- 共有資産とは、例外なく全てのプレイヤーが所有権を持つ資産を指します。

<hr/>

- PMとは、`/tell` コマンドを用いて送信可能なプライベートメッセージを指します。
- AFKとは、一定時間サーバーにログインしたまま席を離れることを指します。
- TTとは、敵対モブまたはそれに準ずるものを一箇所に集め、自動ないしは手動で倒し、アイテムや経験値を入手することを目的とした施設を指します。

## 利用資格

本サーバーをご利用いただける方は、以下の条件を満たす方に限ります。
- 日本語の通じる方
- Minecraft Java Edition あるいは Bedrock Edition（統合版）の正規版をご利用の方
- 他のアカウントでBANされていない方

## スタッフ

スタッフとは、オーナーおよび、それが任命する特別なプレイヤーを指します。スタッフからなるチームをスタッフチームと呼び、別名「運営」「XelticaMC スタッフチーム」などと呼称されます。

オーナー、モデレーター、レポーター、クリエーター、デベロッパー、イベンター、ガイドの7つのロールがあり、各ロールごとに別途規約が適用されます。

スタッフには、本サーバーの運営を円滑に進めるために働く責務が課せられます。

## 共有ワールド

以下のワールドは共有ワールドとなります。共有ワールドにある資産（ブロック、チェスト内のアイテム、モブなどの全ての要素）は共有資産となります。
* ワイルドエリアA
* ワイルドエリアB
* ワイルドネザー
* ワイルドエンド
* クラシックワイルドエリア

共有ワールドで一時的に資産を保護するために保護機能を使うことは認めますが、保護機能を使っていなかったことによる窃盗などがあっても一切責任を負いかねます。

## 自治体

* **自治体**とは、プレイヤーが所有権を持ち、広範に渡り、スタッフの承認を得たものを指します。
* 自治体を申請した者を、その自治体の**代表者**と呼びます。また、自治体は代表者が所有権を持つ個人資産です。

## 協会

* スタッフと協力し、一定の目的を達成するための、特殊な権限を持った市民の集まりを協会と定義します。
* 協会に属する市民を**会員**と、協会のリーダーを**会長**と定義します。
* 協会に入退会する場合は、必ず会長にその旨を通達します。
* 協会は、モデレーターが市民との協議のもと設立します。
* 協会は、モデレーターが特別定める会長と、モデレーターグループによって共同で運営します。

## 禁止行為

### 財産について

- **グリーフィング:** 所有権のない資産を意図的に毀損することは禁止されます。
	- これに反した場合、第一処罰を警告、第二処罰をBANとします。
- **窃盗:** 所有権の無い資産を断りなく盗む・使用することは禁止されます。
	- これに反した場合、第一処罰を警告、第二処罰を没収およびBANとします。
- **共有資産独占:** 共有資産を己の個人資産として独占することを禁止します。
	- これに反した場合、第一処罰を警告、第二処罰を没収およびBANとします。
	- ただし、共有ワールドのTTを除く資産の一部を保護したり、他ワールドに持ち込んで保存することは、これにあたりません。
- **無許可での私有地侵入:** 所有権の無い土地へ侵入することは禁止されます。
  - これに反した場合、第一処罰を警告、第二処罰をBANとします。
  - 所有者の許可を得ている場合はこの限りではありませんが、その場合は所有者の指示に従う必要があります。
  - 所有者の指示に従わなかった場合、第二処罰をその土地への進入禁止とする場合があります。

### 鉄道について

- **鉄道敷設:** メインワールドにおいて、私有地の外に鉄道を敷設する場合は、鉄道協会に入会し鉄道協会長の承認を得なければなりません。
  - 敷設者が所有権を持つ私有地内に作る場合はこの限りではありません。
  - これに反する場合、強制撤去とします。
- **鉄道協会長の義務:** 鉄道協会長には、以下の義務が発生します。
	- 鉄道を管理する義務
	- 鉄道が自治体を侵害しないようにする義務
	- 管理外の鉄道の存在をスタッフに通報する義務
	- これらの義務を果たせない場合、モデレーターはいつでも会長を解任することができます。

### コミュニティについて

- **コミュニティガイドライン違反:** コミュニティガイドラインに反応するような行為を一律禁止します。
	- 詳しくは[コミュニティガイドライン](//wiki.craft.xeltica.work/community-guideline)をお読みください。
	- これに反した場合、第一処罰を警告、第二処罰をその内容によってBANまたはミュートとします。
- **取り決め無きPvP:** 相手の意思に反し、一方的にPvP行為を行うことを禁止します。
	- これに反した場合、程度により警告、またはBANとします。
- **公序良俗に反するコンテンツの作成:** 公序良俗に反すると判断されるコンテンツの作成は禁止されます。
	- これには、猥褻物、違法薬物、暴力といったものが含まれます。
	- これに反した場合、第一処罰を警告、第二処罰を強制撤去とします。
	- なお、繰り返しの違反が確認される場合、第一処罰をBANとする場合があります。
- **恐喝:** いかなる理由においても、ゲーム内通貨やアイテムを脅し取ろうとする行為は禁止します。
	- これに反した場合、BANします。
- **PMの晒し上げ行為:** 他プレイヤーからのPMを晒し上げる行為を禁止します。
	- これに反した場合、第一処罰を警告、第二処罰をBANとします。
	- なお、運営からの警告であるPMの場合は、第一処罰をBANとします。

### 不正行為について

- **資産の現実での取引:** ゲーム内通貨やアイテム等を、現実の財産によってユーザー間で取引することを禁止します。
	- これに反した場合、BANします。
- **共謀:** 規約違反とされる行動をtellコマンド等で共謀することを禁止します。
  - これに反した場合、BANします。
- **意図的に負荷を掛ける行為:** サーバーに意図的に負荷をかけるような行為を禁止します。
	- これに反した場合、BANします。
- **利益目的のAFK:** 利益を得ることを目的としたAFK行為を禁止します。
	- これに反しても処罰はありません
	- AFK によって発生した問題について、運営は一切の責任を負いかねます。
- **利益目的でのサブアカウント運用:** 利益を得ることを目的としたサブアカウント運用を禁止します。
  - これに反した場合、メインアカウントを含め全てBANします。
- **無許可のTT作成:** TTは許可制となっております（下記）。許可せずに、また許可申請通りでないTTの作成・運用を禁止します。
  - これに反した場合、強制撤去します。
- **不具合悪用:** プラグインや権限設定、マイクラ自身のバグなど、あらゆる不具合の悪用・および他プレイヤーへの流布を禁止します。
	- これに反する場合、悪質度によって適切とされる処罰を行います。
	- なお、報告のために必要な範囲内で不具合を再現する場合、「悪用」に該当しません。
	- 不具合を発見した場合は、なるべく迅速な報告をお願いします。
- **処罰回避:** あらゆる手段で、処罰を回避するためにアカウントを作成し、ログインしようとする行為を禁止します。
	- これに反した場合、BANします。
- **虚偽通報:** 運営に対し、虚偽の通報を意図的に行うことを禁止します。
	- ただし、勘違いに関しては「意図的」とみなしません。
	- これに反した場合、BANします。
- **運営からのPMの無視:** 運営からの聴取を無視し、答えないことを禁止します。
	- これに反した場合、聴取のきっかけとなった行為を元に処罰されます。
- **不正MODの使用:** ゲームを不法に有利に進める目的でいわゆるチートMOD系を導入するプレイを禁止します。
	- これに反した場合、第一処罰を警告、第二処罰をBANとします。
	- 該当するMODの例は、下記の「MODについて」を参照。
- **アカウント共有:** 複数人で同じアカウントを用いて参加する行為を禁止します。
	- これに反した場合、第一処罰を「キック」、第二処罰をBANとします。
- **運営を妨害する行為:** スタッフによる運営を妨害する行為を一律禁止します。
  - これには、スタッフを騙る行為、スタッフがすべき仕事（規約を読んだかの確認）などが当たりますが、これに限りません。
  - これに反した場合、悪質な場合はBANします。
- **年齢詐称行為:** 市民ロールへの移行時に、意図的に不正な生年月日を入力することを禁止します。
  - これに反した場合、BANします。
- **協会会員による権限の悪用:** 協会に与えられた権限は、スタッフが協会に与えた業務の遂行にのみ使用を認めており、その他用途外の利用を全て禁じます。
  - これを悪用した場合、第一処罰を警告とし、第二処罰を当該協会からの強制脱会とします。


その他、プレイヤー間でのトラブルを起こしてはいけません。トラブルを起こした場合、トラブルの発端はBANされます。

## TT建設について

本サーバーでは、ゲームバランス確保とサーバー負荷管理のために、TT建設ならびに運用を許可制としています。

TTを運用する場合、Discordサーバーの #申請所 チャンネルで申請を行ってください。

ただし、TT申請には次の条件があります。

* 装置を停止可能な機構が付いていること
* 似たような装置が無いこと
* 全てのプレイヤーに公開すること
* ワイルドエリアA, ワイルドネザー, ワイルドエンドのいずれかであること
  * ただし、クラシックワールドに限り、これまで建てられたTTは不問としています
* その他、運営内部基準（非公開）を満たすこと

TT申請時は次のテンプレートに添って、括弧で書かれた場所を置き換えてください（カッコごと置き換える）。

```
名前: (簡単な名前(例: ゾンビTT))
ワールド名: (ワールド名(例: メインワールド))
座標: (X), (Y), (Z)
目的: (目的を簡潔に書いてください)
作り方: (作り方を詳しく書くか、参考となるURLを貼ってください)
```

お手数をおかけしますが、何卒よろしくおねがいします。

### お願いしたい事柄

- 土地を確保する場合は、目印と共に立て札を建てて明示するか、保護プラグインの機能を用いて保護してください
	- 明瞭な状態にしていない場合、土地は個人資産とはみなされません
- 燃えやすい素材のすぐ近くに炎・溶岩等の炎上の原因となるブロックを設置しないよう心がけてください
- 畜産目的でのモブ小屋は、脱走防止のために必ず戸締りをしてください
	- 意図的に開けたまま放置した場合、破壊禁止違反でBANされます
- 作物や原木を採取した後は、必ず同様の作物・苗木を植え直してください
- 建築のため、メインワールドでのブランチマイニングはしないでください
	- 代わりにワイルドエリアを使用してください
- 景観保護の為、メインワールドにおいて、次のような事柄は避けてください。
	- 無意味にブロックを積み上げたまま放置
	- 穴を開けて放置
	- 日照権を侵害する大型建築
	- これらに反する場合、強制撤去します。
- 動画の撮影・ライブ配信などを行う際は、X Phoneから「配信モード」を有効化していただくようお願いします。
- 本サーバーでの活動やスキン、創作物は、第三者の動画撮影やライブ配信等に映り込む可能性があることをご理解お願いします。
- 統合版サポートは未だ試験段階であることをご理解願います。

### MODについて

XelticaMCでは、ゲームプレイを支援するクライアントMOD、リソースパックなどの導入を許可しています。
しかし、以下に示すようなMODは一切の使用を禁止します。これに反した場合、第一処罰を警告、第二処罰をBANとします。

- Xray などの、一部ブロックのみを透けて見えるようにするMOD
- 飛行MOD
- 高速移動MOD
- 一括破壊MOD
- ノックバックを防止するMOD
- オートエイムMOD
- 連射ツール
- その他、一般的にチートと判断されるMOD

以下のようなMODは使用を許可しています。

- パフォーマンス改善MOD, 影MOD
- マップMOD(洞窟マップもOK)
- インベントリ整理MOD
- レシピ確認MOD
- スポーンチェッカーMOD
- HUD追加MOD
- 音響効果を拡張するMOD

場合によっては、お使いのMODがチートとみなされる可能性があります。もし不安でしたら、ゲーム内チャットやDiscordにて直接お問い合わせください。

## 警告が不可能な場合

違反を確認した場合に警告対象のユーザーがログインしていない場合、アカウントに紐づけてある、あるいは運営が把握しているDiscordアカウントにダイレクトメッセージで警告を行うか、市民認証時のメールアドレスに警告を行います。

特にxeltica@gmail.comからのメールアドレスは何らかの形で確認可能にしておくことを固く推奨します。

上記のいずれも警告手段として用いることができない場合、警告処罰は不可能な為スキップします。

## マナー

本サーバーにて、守るべきマナーをまとめます。

- **なるべく挨拶すること**
  - 特に新規さんは挨拶がないだけで怪しいプレイヤーという疑いをかけられる可能性があります
- 1対1の会話が続く場合、tellを使うこと
  - 他プレイヤーの邪魔になりうるチャットの使い方は、ミュート対象となる場合があります
- tellにはtellで返すこと
  - [相手→自分] という文字列から始まるチャットは、あなたにしか見えません。
  - `/tell ユーザー名 内容` でtellを行えます
- チャット欄になるべく目を通すこと
  - お知らせや運営からの警告はチャット欄に表示されます
  - 特に警告を無視すると処罰対象となるため、注意してください
- 違反プレイヤーを発見した際は、[専用フォーム](https://forms.gle/egnTF6kMNTyiRHrv9)にて通報すること。
  - チャットで騒ぐ行為は、違反プレイヤーの思う壺である可能性が高いです。騒がずに報告してください。

また、[コミュニティガイドライン](//wiki.craft.xeltica.work/community-guideline)もお読みください。
